ID e CLASS

ID

O ID funciona como um CPF, assim como só pode ter um numero de CPF, cada elemento só pode ter um valor de ID. On CPC é unico, ou seja, ninguem pode ter um CPF igual ao meu, isso significa que o ID tambem é unico, então não é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID, diferente do Id que você só pode ter um valor, no ClASS você pode ter varios valores. Também diferente do ID, em que não se pode repetir um valor, no CLASS você pode utilizar o mesmo valor para varios elementos.
